## Descripción
Este proyecto corresponde al Ejercicio 10 de la Práctica 2. Se simula una competencia de cocina donde cinco participantes son evaluados por tres jueces a través de cinco rondas.

En cada ronda: se calcula el puntaje total de cada participante, se determina el ganador y se actualiza la tabla de posiciones

Al final:
- Se muestra una tabla final con: puntaje total acumulado, cantidad de rondas ganadas, mejor puntaje en una ronda y promedio por ronda
